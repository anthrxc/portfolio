Who are you?

Thank you for asking!

I'm an 18-year-old freelance developer from Croatia.

I always had an interest in creating something of my own and sharing it with the world, so programming was the perfect combination with my existing interest in technology. My coding journey started around the time of COVID lockdown, when I couldn't do much apart from sit at home and cope with online school.

Since then, I have turned many of my weird and (at least for me) interesting ideas into reality with JavaScript. Later, I started experimenting with website design and development which is also an interesting endeavour for me. Besides those, I am proficient in creating and maintaining SQL or MongoDB databases, so whatever it is that a project requires, I can assure you I am capable of it.

When I'm not working on any projects, you'll either find me in a gym, trying to get stronger, or behind a book, trying to escape the education system.
Other than those, I like to watch Formula 1 whenever it's a race weekend, or simracing when it isn't.

I'm available for hire! If you have a project in mind or just want to connect, feel free to drop me a message using the contact form at the bottom of the page.

EuroFinance, Ltd.

This website was created as part of a school project, where we were assigned to build a site based on various topics. Over the next few days, I invented a fictional bank and designed a website for it. The site included multiple pages introducing visitors to the bank, its services, and the fictional team behind it. My professor praised the website as the best in the class.

Bela+

Bela+ is a program designed to track points in the game of Belote. I developed this program for two reasons: to track points when playing the game with friends, and to submit it as my entry for a programming competition. Although the competition outcome wasn't as I had hoped for reasons beyond my control, the program itself was great. Bela+ was developed in JavaScript using the 'enquirer' package and compiled into an executable with 'nexe'.

Passman CLI

Passman CLI began as a hobby project to create my own password manager and introduced me to the 'enquirer' library, later used in Bela+. Initially, it was an app to track assignments and classes for a friend, but I soon realized it could be adapted to track almost anything, so I developed it into a password manager. Building this project was a fascinating learning experience, and I was proud of the final result. Although I attempted to develop a desktop app version, schoolwork eventually took priority, and the project was shelved.

Lott

Lott was a JavaScript-based Discord bot where users could send cryptocurrency to enter a "lottery". The bot featured server-specific lotteries with weekly draws and a bot-wide lottery with monthly draws. Unfortunately, although the bot was fully developed, issues with the API handling cryptocurrency deposits prevented it from going live.

Oversight

Oversight was a Discord bot developed to moderate and monitor activities on my old Discord server. It combined the features of a moderation bot, server logging tool, and ticketing system. This project was quite cool to me because allowed me to create a bot with all the necessary functions, without relying on the paywalled features often found in similar bots.

SineWave

SineWave was a Discord bot inspired by the Rick Astley bot, which rickrolled a voice chat whenever someone @mentioned the bot. However, instead of playing "Never Gonna Give You Up," SineWave played a 6300 Hz sine wave tone for 30 seconds. It was a fun, short-lived project and an excellent one-day exercise to learn the basics of Discord's voice API.
